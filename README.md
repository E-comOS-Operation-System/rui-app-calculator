## How to install it
E-comOS are using EPM(E-comOS Packages Manager)to manager our packages , Mr.Mangaer maybe DO NOT like we skip to he to install software (Emm...RUI GUI Install is using EPM,too) , so you must to use EPM to install this software .
But in general , this software is installed in your computer , Unless you're using a derivative version or you simply ran the following when installing RUI:

```bash
epm install rui
```

Instead of

```bash
epm install @RUI-GOUPS
# or
epm install rui rui-app-calculator rui-app-clock rui-keyboard-English (For example) ...
```
To install this app , you can use this command in your terminal :
```bash
epm install rui-app-calculator
```
**IF YOUR COMPUTER IS ISN'T RUNNING OF E-comOS , YOU CANNOT USE THIS APP , BECAUSE THIS APPS IS ISN'T USING UNIX-Like IPC MESSAGES SYSTEM**
## How to use it
After you installed it , you can find it in RUI MALL or showcase(Use your Mark key , its 'Win' key on Windows and 'Super' Key in Linux).
Click it to open it , and you can use it now!
